# QuickBlog

Hola!

Esta es una pequeña aplicación como ejemplo de CRUD con conexion a base de datos, firebase en este caso.
Permite crear usuario, loguearse y crear entradas por cada usuario registrado, el cual podrá eliminar y actualizar sus
propios posts, filtar la lista de posts por los parametros dados inicialemnte (toas las entradas, todas menos las propias,
solo las propias), como esta app es un MVP realizado en tiempos libres, es muy sencilla pero esta pensada para tener caracteristicas 
como comentarios en los posts, agregar imagenes, listas, listas de tareas etc... Y hacer black o whitelists para los 
permisos de visualizacion y creacion de comentarios.

Esperen las siguientes versiones!!
Se planean muchas mejoras y mas características!!
